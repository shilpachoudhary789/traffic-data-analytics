# Traffic Data Analytics Project

This project analyzes traffic datasets to identify congestion patterns and peak traffic hours.

## Tools Used
SQL  
Python  
Power BI  
Tableau  

## Dataset
The project uses traffic data including:
- Traffic volume logs
- Signal timing configuration
- Public transport delay records

## Project Steps
1. Data exploration and cleaning using Python.
2. Data analysis using SQL queries.
3. Identifying traffic congestion patterns.
4. Building dashboards in Power BI and Tableau.

## Key Insights
- Identified peak-hour traffic congestion patterns.
- Analyzed traffic flow across different signals.
- Built dashboards to visualize traffic trends and support data-driven decision making.
